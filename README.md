# Hardware E-Commerce Store

## Project Documentation

Welcome to the Hardware E-Commerce Store project. This document provides an overview of the project, its structure, and how to set it up and use it.

## Overview

This project is designed to create an online platform for selling hardware components. It allows users to browse products, add them to their cart, and make purchases.

## Features
- User Authentication
- Product Browsing
- Shopping Cart
- Secure Payment Processing
- Order History

## Getting Started

### Prerequisites
- Node.js and npm installed
- MongoDB setup

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dccv542-ctrl/Hardware-
   cd Hardware-
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```

### Usage
- Open your browser and navigate to `http://localhost:3000`
